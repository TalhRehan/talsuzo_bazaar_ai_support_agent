# Project Identity

## Name

TalSuzo Bazaar

## Package Name

`talsuzo-bazaar`

## Project Type

Full-stack ecommerce website with an AI-powered customer support agent for refund decisions.

## Purpose

TalSuzo Bazaar allows customers to browse products, manage cart and checkout flows, and submit refund requests through an AI support interface. The AI support system must evaluate requests using customer data, order history, and official refund policy rules.

## Core Areas

- Ecommerce storefront
- Customer refund support chat
- FastAPI backend
- LangGraph AI agent workflow
- Refund policy engine
- Synthetic CRM and order data
- Admin dashboard for decision logs
- Docker Compose local setup

## Branding Rules

Use `TalSuzo Bazaar` for the visible business name.

Use `talsuzo-bazaar` for package, repository, Docker service, and technical identifiers.

Do not use any previous storefront branding in visible UI, documentation, or project metadata.
